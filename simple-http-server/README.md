# simple-http-server

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.tsx
```

To watch:

```bash
bun run --watch index.tsx
```

This project was created using `bun init` in bun v1.0.22. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
